# Retail Sales Analysis using SQL

## Project Overview

This project analyzes four years of retail sales transactions from a global superstore using SQL. The objective is to transform raw transactional data into meaningful business insights that support data-driven decision-making.

The project simulates the work of a Data Analyst responsible for evaluating sales performance, customer behaviour, product profitability, and regional performance.

---

## Business Scenario

ABC Retail is a multinational retail company operating across multiple regions.

The management team wants answers to questions such as:

- Which products generate the highest revenue?
- Which customers contribute the highest sales?
- Which regions perform best?
- Which product categories are losing money?
- How do discounts affect profitability?

As a Data Analyst, my responsibility is to answer these questions using SQL.
