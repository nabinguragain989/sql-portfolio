# SQL Portfolio | Data Analytics

Welcome to my SQL Portfolio! 👋

This repository showcases my SQL skills through real-world business case studies and end-to-end data analysis projects. The goal of this portfolio is to demonstrate my ability to write efficient SQL queries, analyze complex datasets, solve business problems, and communicate insights that support data-driven decision making.
